/etc/openhab2/items
===================

* Benötigte Items können problemlos, wenn gewünscht, auf (beliebig viele) Dateien mit Endung `.items` verteilt werden
* Änderungen über einen der unterstützten Editoren vornehmen

Format
------

```
itemtype itemname
  "itemlabel [stateformat]"
  <iconname>
  (group1, group2, ...)
  ["tag1", "tag2", ...]
  {bindingconfig}

```
